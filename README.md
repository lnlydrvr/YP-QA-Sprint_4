# Приложение BooksCollector - 4 спринт автоматизированного тестирования на Python

Для проведения тестов используется библиотека `pytest`.

## Метод `add_new_book`

### 1. Тест добавления книги с корректным именем
Тест проверяет, что книга с корректным именем успешно добавляется в коллекцию. Если книга была добавлена, её имя должно присутствовать в словаре books_genre.

- Название теста: `test_add_new_book_with_valid_name_book_has_been_added`
- Ожидаемый результат: Имя книги присутствует в `collector.books_genre`.

### 2. Тест проверки жанра для добавленной книги с корректным именем
Этот тест проверяет, что при добавлении новой книги с корректным именем, жанр книги по умолчанию остается пустым.

- Название теста: `test_add_new_book_with_valid_name_book_genre_is_empty`
- Ожидаемый результат: Жанр книги, добавленной в `collector.books_genre`, пустой (пустая строка).

### 3. Тест добавления книги с некорректным именем
Тестирует сценарий, когда пытаются добавить книгу с некорректным именем. В этом случае книга не должна быть добавлена в коллекцию.

- Название теста: `test_add_new_book_with_invalid_name_book_has_not_been_added`
- Ожидаемый результат: Имя некорректной книги отсутствует в `collector.books_genre`.

### 4. Тест добавления книги с повторяющимся именем
Тест проверяет, что если попытаться добавить книгу с тем же именем дважды, то она не будет дублироваться в коллекции.

- Название теста: `test_add_new_book_with_same_name_twice_book_is_not_duplicated`
- Ожидаемый результат: Имя книги присутствует в `collector.books_genre` только один раз.

## Метод `set_book_genre`

### 1. Тест присвоения корректного жанра существующей книге
Тест проверяет, что метод `set_book_genre` корректно присваивает заданный жанр книге, которая уже существует в коллекции.

- Название теста: `test_set_book_genre_with_valid_genre_and_an_existing_book_genre_has_been_assigned`
- Ожидаемый результат: Жанр книги в `collector.books_genre` соответствует ожидаемому значению.

### 2. Тест присвоения некорректного жанра существующей книге
Тест проверяет, что если попытаться присвоить некорректный жанр существующей книге, то жанр не будет изменён и останется пустым.

- Название теста: `test_set_book_genre_with_invalid_genre_and_an_existing_book_genre_has_not_been_assigned`
- Ожидаемый результат: Жанр книги в `collector.books_genre` остаётся пустым, если был передан некорректный жанр.

### 3. Тест присвоения корректного жанра несуществующей книге
Этот тест проверяет, что если попытаться присвоить жанр книге, которой нет в коллекции, то ни книга, ни жанр не будут добавлены в коллекцию.

- Название теста: `test_set_book_genre_with_valid_genre_and_an_non_existent_book_has_not_been_added_and_genre_has_not_been_assigned`
- Ожидаемый результат: Книга с заданным именем не появляется в `collector.books_genre`, если она не была добавлена ранее.

## Метод `get_book_genre`

### 1. Тест получения жанра существующей книги
Тест проверяет, что метод `get_book_genre` возвращает правильный жанр для книги, которая уже добавлена в коллекцию и имеет установленный жанр.

- Название теста: `test_get_book_genre_of_an_existing_book_genre_is_returned`
- Ожидаемый результат: Метод возвращает ожидаемый жанр, который был установлен для книги.

### 2. Тест получения жанра несуществующей книги
Тест проверяет, что метод `get_book_genre` возвращает `None` для книги, которая не была добавлена в коллекцию.

- Название теста: `test_get_book_genre_of_an_non_existent_book_genre_is_not_returned`
- Ожидаемый результат: Метод возвращает `None`, если книга не существует в коллекции.

### 3. Тест получения жанра существующей книги без установленного жанра
Тест проверяет, что метод `get_book_genre` возвращает пустую строку для книги, которая была добавлена в коллекцию, но для которой жанр не был установлен.

- Название теста: `test_get_book_genre_of_an_existing_book_without_genre_genre_is_empty`
- Ожидаемый результат: Метод возвращает пустую строку, если жанр для книги не был установлен.

## Метод `get_books_with_specific_genre`

### 1. Тест получения книг по заданному жанру
Тест проверяет, что метод `get_books_with_specific_genre` возвращает список книг, которые имеют заданный жанр, при условии что жанр был правильно установлен.

- Название теста: `test_get_books_with_specific_genre_genre_is_assigned_and_an_existing_books_books_are_returned`
- Ожидаемый результат: Метод возвращает список книг, которые имеют указанный жанр, если жанр был корректно установлен.

### 2. Тест получения книг по неверному жанру
Этот тест проверяет, что метод `get_books_with_specific_genre` возвращает пустой список, если для существующих книг установлен правильный жанр, но запрашивается жанр, которого нет.

Название теста: `test_get_books_with_specific_genre_an_existing_books_and_wrong_genre_books_are_not_returned`
- Ожидаемый результат: Метод возвращает пустой список, если запрашивается жанр, который не совпадает с установленным жанром книг.

### 3. Тест получения книг по несуществующему жанру
Тест проверяет, что метод `get_books_with_specific_genre` возвращает пустой список для жанра, который не был назначен никакой книге в коллекции.

- Название теста: `test_get_books_with_specific_genre_an_existing_books_and_invalid_genre_books_are_not_returned`
- Ожидаемый результат: Метод возвращает пустой список, если запрашивается недопустимый жанр.

## Метод `get_books_genre`

### 1. Тест получения пустого словаря для пустой коллекции
Этот тест проверяет, что метод `get_books_genre` возвращает пустой словарь, если в коллекции нет ни одной книги.

- Название теста: `test_get_books_genre_books_not_added_return_empty_vocabulary`
- Ожидаемый результат: Метод возвращает пустой словарь, когда в коллекции нет книг.

### 2. Тест получения словаря с книгой без жанра
Тест проверяет, что если книга была добавлена в коллекцию без указания жанра, то метод `get_books_genre` возвращает словарь, где ключом является имя книги, а значением — пустая строка.

- Название теста: `test_get_books_genre_book_is_added_without_genre_return_only_book_name_in_vocabulary`
- Ожидаемый результат: Метод возвращает словарь, где book_name — имя добавленной книги, а значение — пустая строка, указывающая, что жанр не установлен.

### 3. Тест получения словаря с книгой и установленным жанром
Этот тест проверяет, что если книга была добавлена и для неё установлен жанр, метод `get_books_genre` возвращает словарь, где ключом является имя книги, а значением — установленный жанр.

- Название теста: `test_get_books_genre_book_is_added_with_genre_return_book_name_and_genre_in_vocabulary`
- Ожидаемый результат: Метод возвращает словарь с указанием названия книги и её жанра.

## Метод `get_books_for_children`

### 1. Тест получения книг для детей из коллекции с книгами для детей и взрослых
Этот тест проверяет, что метод `get_books_for_children` возвращает только те книги, которые относятся к детской категории, игнорируя книги, подходящие только для взрослых.

- Название теста: `test_get_books_for_children_with_child_and_adult_book_return_only_child_book`
- Ожидаемый результат: Метод возвращает список книг для детей, в который включается только книга, имеющая жанр, подходящий для детей.

## Метод `add_book_in_favorites`

### 1. Тест добавления существующей книги в избранное
Этот тест проверяет корректность работы метода `add_book_in_favorites` для книги, которая уже присутствует в коллекции.

- Название теста: `test_add_book_in_favorites_an_existing_book_book_is_added_in_favorites`
- Ожидаемый результат: Метод корректно добавляет книгу в список избранного, и она появляется в коллекции избранного.

### 2. Тест попытки добавить несуществующую книгу в избранное
Этот тест проверяет, что если попытаться добавить книгу, которая не была добавлена в коллекцию, метод `add_book_in_favorites` не добавляет её в список избранного.

- Название теста: `test_add_book_in_favorites_an_non_existing_book_book_is_not_added_in_favorites`
- Ожидаемый результат: Метод не добавляет несуществующую книгу в список избранного, и книга не появляется в коллекции избранного.

## Метод `delete_book_from_favorites`

### 1. Тест удаления книги из избранного
Этот тест проверяет, что метод `delete_book_from_favorites` правильно удаляет книгу из списка избранного, если книга была ранее добавлена в этот список.

- Название теста: `test_delete_book_from_favorites_book_in_favorites_book_deleted`
- Ожидаемый результат: После вызова метода указанная книга больше не присутствует в списке избранного.

## Метод `get_list_of_favorites`

### 1. Тест получения пустого списка избранного
Этот тест проверяет, что метод `get_list_of_favorites` возвращает пустой список, если в коллекции нет книг, добавленных в избранное.

- Название теста: `test_get_list_of_favorites_books_books_is_not_added`
- Ожидаемый результат: Метод возвращает пустой список, когда в списке избранного нет книг.

### 2. Тест получения списка избранного после добавления книги
Тест проверяет, что метод `get_list_of_favorites` возвращает список, содержащий книги, которые были добавлены в избранное. Если книга была добавлена в коллекцию и отмечена как избранная, она должна отображаться в возвращаемом списке.

- Название теста: `test_get_list_of_favorites_books_books_is_added`
- Ожидаемый результат: Метод возвращает список с книгой, если книга была добавлена в избранное.
